DEAD ROUTE — Audio Files
=========================

Drop your audio files into these folders. Supported formats: .mp3, .ogg, .wav, .flac, .m4a

MUSIC FOLDER (audio/music/):
  title.mp3           — Title screen / boot music
  intro.mp3           — Intro text crawl
  morning.ogg         — Morning phase (hopeful, cautious) — plays days 1-9
  day_10_morning.ogg  — Morning phase from day 10+ (darker, the hope is fading)
  afternoon.mp3       — Afternoon phase (tension building)
  evening.mp3         — Evening phase (dark, foreboding)
  midnight.mp3        — Midnight phase (eerie, terrifying)
  combat.mp3          — Combat encounters
  event.mp3           — Random event / narrative moments
  horror_ambient.mp3  — Eerie silence — plays during infection, turning, dark events
  haven.mp3           — Haven ending (bittersweet/triumphant)
  gameover.mp3        — Game over / death
  travel.mp3          — Traveling between nodes

SFX FOLDER (audio/sfx/):
  engine_start.mp3  — Bus engine starting
  gunshot.mp3       — Euthanasia / combat shots
  bite.mp3          — Infection bite
  upgrade.mp3       — Bus upgrade installed
  loot.mp3          — Finding resources

Music loops automatically. SFX play once.
All filenames are configurable in engine/audio.py.
